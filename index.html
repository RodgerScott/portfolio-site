About

My name is Rodger Scott: I am a San Antonio native, proud father, and a very lucky husband. After over a decade as a professional in the educational assessment industry, I decided to follow my dreams of working in the world of software development and information technology. My previous career path has given me a deep background in editing, management, and customer service.

I am super stoked to be a part of the San Antonio tech community after many years of admiring it from afar. I always try to live in the moment, not take things for granted, and utilize new opportunities as a way to grow both personally and professionally.

Projects

Pig Latin Converter

Built with: HTML5, CSS, Javascript

Having three little ones running around, there are some times when you need to communicate in code. This inspired my first app which was developed after my first month of coding. This project introduced me to regular expressions in addition to basic conditional logic.

Simon

Built with: HTML5, CSS, Javascript, Jquery

Ahh Simon... the classic game of memory. I though it would be fun to try and recreate the classic look of the Simon game using CSS. This project introduced me to the asynchronous nature of Javascript as I needed to incorporate timeouts in order to keep the game progressing as it should (sans-wonkiness).

Tic-Tac-Toe

Built with: HTML5, CSS, Javascript, Jquery

After completing a codewars.com challenge to create scoring logic for a tic-tac-toe game, I figured I might as well create the actual tic-tac-toe game itself. Creating the basic board allowed me to practice working with 2d arrays, jQuery event methods, and I was able to apply a nested loop to come up with the "it's a draw" conclusion.
